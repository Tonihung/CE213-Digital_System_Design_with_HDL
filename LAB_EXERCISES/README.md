# Laboratory Exercises

This directory contains the laboratory exercises for the **Digital System Design with HDL** course.

Each laboratory folder includes Verilog HDL source files, testbenches, and simulation materials.

## Laboratory List

- [LAB02](./LAB02) – Synchronous Counter and Register File
- [LAB03](./LAB03) – Finite State Machine and Data Memory
- [LAB04](./LAB04) – Arithmetic Logic Unit and Multiplexer
- [LAB05](./LAB05) – Simple MIPS Datapath
- [LAB06](./LAB06) – Simple MIPS Control Unit
- [LAB07](./LAB07) – Simple MIPS Processor

> Detailed descriptions, implementations, and simulation instructions are provided in the README file of each laboratory folder.
